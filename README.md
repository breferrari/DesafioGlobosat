# DesafioGlobosat

## Prerequisitos

### Cocoapods Setup
```sh
sudo gem install cocoapods
pod install
```

## Acesso

### Login e senha
```sh
Login: desafio
Senha: root
```

## Dependências usadas
- Contentful (JSON)
- Realm (ORM/Persistência)
- SDWebImage (Cache e async load de imagens)
- SVProgressHUD (UI, Indicador de atividade)
